- **Projetos e Práticas**

  - Exercícios resolvidos

- **Tópicos Avançados**

## Estrutura

Cada pasta representa uma categoria de estudo, com arquivos nomeados de forma clara e sequencial. O objetivo é facilitar a navegação e a revisão do conteúdo.

## Objetivo

Registrar meu progresso no aprendizado de Python e manter um histórico consultável e versionado dos meus estudos.
